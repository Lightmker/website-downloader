### How to run it on Termux/Vps/Pc
- Copy paste these commands one by one:
```
git clone https://github.com/mouricedevs/website-downloader.git
```
```
cd website-downloader
```
```
npm install
```
```
npm start
```
- Open your browser and paste: 
```
http://localhost:3000
```
